# ONYX Hotel Management System

Полноценная система управления гостиницей с клиентской и административной панелями. Разработана на Flask с использованием SQLite в качестве базы данных.

## Возможности

### Клиентская часть

- Регистрация и авторизация клиентов с проверкой данных
- Просмотр номеров с фильтрацией по категориям и статусу
- Поиск свободных номеров по датам заезда/выезда и количеству гостей
- Бронирование номеров с автоматическим расчетом стоимости
- Предоплата 30 процентов при бронировании
- Личный кабинет с историей всех бронирований
- Отмена бронирования (бесплатно за 24 часа до заезда)
- Детальная страница номера с описанием, удобствами и правилами
- Контактная страница с FAQ и правилами проживания

### Административная панель

- Дашборд с ключевыми метриками: количество клиентов, номеров, бронирований
- Статистика загрузки номеров по категориям
- Ближайшие заезды (сегодня и завтра)
- Финансовая статистика (предоплаты, остатки к оплате)
- Популярные номера на основе количества бронирований
- Свежие бронирования в реальном времени
- Управление номерами: добавление, редактирование, удаление
- Управление бронированиями со сменой статусов
- Управление клиентами с поиском по фамилии, имени, телефону
- Отчеты и аналитика с графиками бронирований по месяцам

## Технологии

- Backend: Python 3.x, Flask
- Database: SQLite3
- Frontend: Bootstrap 5, Font Awesome 6
- Charts: Chart.js для графиков
- Templates: Jinja2
- Styling: Custom CSS с черно-зеленой темой оформления

## Возможные проблемы и решения
### Проблема: Ошибка "База данных не найдена"

- Решение: Запустите команду python db.py для создания базы данных и заполнения её тестовыми данными.

### Проблема: Ошибка при загрузке изображений номеров

- Решение: Убедитесь, что папка static/images/rooms/ существует и имеет права на запись. Проверьте, что файлы изображений названы в соответствии с правилами именования.

### Проблема: Порт 5000 уже занят другим приложением

- Решение: Измените порт в файле app.py, заменив app.run(debug=True, host='0.0.0.0', port=5000) на другой порт, например 5001.

### Проблема: Ошибка внешнего ключа при попытке удалить номер

- Решение: Сначала удалите или измените статус связанных бронирований, после чего повторите попытку удаления номера.

### Проблема: Сессия пользователя истекает слишком быстро

- Решение: Добавьте в app.py настройку: app.config['PERMANENT_SESSION_LIFETIME'] = timedelta(days=7) для увеличения времени жизни сессии.
